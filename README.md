# Tobedo

Tobedo is a **simple mobile-first to-do app with a built-in Pomodoro timer**.  
It focuses on **getting work done quickly without clutter or complex navigation**.

## Features

- **Swipe to focus** – Swipe right on a task to start a Pomodoro session.
- **Task → Focus → Complete** – Start a session and finish the task.
- **Calendar task input** – Add tasks using a simple calendar UI.
- **Clean interface** – No tabs, no extra screens.

## Tech Stack

- React Native
- Expo

## Idea

Tobedo removes friction between **planning and doing**.  
Instead of managing tasks and timers separately, you **start focus directly from the task**.
